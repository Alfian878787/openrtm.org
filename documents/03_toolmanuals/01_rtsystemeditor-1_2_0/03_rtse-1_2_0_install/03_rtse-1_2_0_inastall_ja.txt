// Title: インストールおよび起動
#contents

ここでは、RTSystemEditor のインストールおよび起動について説明します。

*** RTSystemEditor のインストール
RTSystemEditor は Eclipse プラグインであるため、 Eclipse 本体および依存している他の Eclipse プラグインをまずインストールする必要があります。
インストールに関しては、[[OpenRTM Eclipse tools のインストール:/node/676]] を参照してください。

*** RTSystemEditor の起動

+ インストール後、 Eclipse を初めて起動すると、以下のような「ようこそ」画面が表示されます。&br;
画面左上の「✖」をクリックし、「ようこそ」画面を閉じます。&br;&br;
#ref(RTCBuilder1.1.2_021.jpg,80%,center)
&br;&br;
+ 以下の画面で、右上の [パースペクティブを開く] ボタンをクリックします。&br;&br;
#ref(RTCBuilder1.1.2_022.jpg,80%,center)
&br;&br;
+ [RT System Editor] を選択し、[OK] ボタンをクリックします。&br;&br;
#ref(RTCBuilder1.1.2_023.jpg,80%,center)
&br;&br;
+ RTSystemEditor が起動します。&br;&br;
#ref(RTCBuilder1.1.2_024.jpg,80%,center)
&br;&br;
