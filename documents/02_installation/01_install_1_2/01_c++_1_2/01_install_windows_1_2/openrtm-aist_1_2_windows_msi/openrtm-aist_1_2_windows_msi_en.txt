//openrtm.org では Windows 用の msi (Microsoft Windows インストーラー)パッケージを提供しています。
//msi パッケージを利用することで、依存ライブラリである omniORB や、RTSystemEditor といったツールも同時にインストールすることができます。
openrtm.org is providing msi (windows installer) package of OpenRTM-aist 1.2.
By using msi package, you can install the required software packages like omniORB (depeding library) and the tools like RTSystemEditor at the same time.

#clear
#contents


** 依存ソフトウェアのインストール
OpenRTM-aistのインストール前に必ずPythonをインストールする必要があります。

- [[Python(2.7、もしくは3.6、3.7):https://www.python.org/downloads/windows/]]

** OpenRTM-aistのインストール
以下からインストーラー(OpenRTM-aist-1.2.x-RELEASE_x86_64.msi)をダウンロードしてインストールしてください。
ただし、''Pythonを32bit版をインストールした場合はOpenRTM-aistも32bit(OpenRTM-aist-1.2.x-***_x86.msi)をインストールする必要があります。Pythonが64bit版の場合はOpenRTM-aistも64bit版(OpenRTM-aist-1.2.x-***_x86_x64.msi)をインストールします。''

- [[OpenRTM-aist 1.2.0:https://github.com/OpenRTM/OpenRTM-aist/releases/tag/v1.2.0]]
