RTC の開発を支援するために、RTCBuilder と RTSystemEditor を提供しております。
RTCBuilder と RTSystemEditor は、Eclipse 統合開発環境のプラグインとして作成された開発ツールです。
そのため、RTCBuilder と RTSystemEditor を使用するには、Eclipse と Java の開発環境をインストールする必要があります。

ここでは、それらのインストール方法について説明します。
