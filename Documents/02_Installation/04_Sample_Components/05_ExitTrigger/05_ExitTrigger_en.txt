**ExtTrigger

***Overview
RT Component This is a sample ExecutionContext that executes processing according to input (event) from outside. Running ExtConsoleIn.bat and ExtConsoleOut.bat will start the sample component. Please start both components and run ExtConnector.bat to connect between the ports of both components.
//***概要
//RTコンポーネント外部からの入力(イベント)により処理を実行する ExecutionContext のサンプルです。
//ExtConsoleIn.bat と ExtConsoleOut.bat を実行することでサンプル・コンポーネントが起動します。
//両コンポーネントを起動、ExtConnector.bat を実行して両コンポーネントのPort間を接続してください。

If the connection between the ports is successful, a menu will be displayed asking which console the ExtConnector is running to proceed with.
Each component advances one cycle at a time according to the value input to this console.
