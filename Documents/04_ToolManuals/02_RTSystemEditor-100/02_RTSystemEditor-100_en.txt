** OpenRTM-aist RT System Editor Introduction 
The [[OMG:http://www.omg.org]] has developed the Robot Technology Component (RTC) specification as a way to increase the efficiency of robot development. Based on this specification, AIST has developed the [[OpenRTM-aist:http://www.openrtm.org/OpenRTM-aist-Tutorial/]]. RT System Editor is one of the development tools included in this architecture. It is used for manipulating RT Components in a real-time graphical interface. It is a plugin for the Eclipse IDE, and interacts seamlessly with other Eclipse plugins.
